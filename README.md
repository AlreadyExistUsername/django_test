# django_test

### Требуется:
Django==1.11.2

### БД используется:
postgresql

### Функционал микроблога:
Главная страница содержит:
 - Поле для ввода текста сообщения
 - Кнопка "отправить"
 - Список всех предыдущих сообщений
 
Регистрация не требуется, чтобы оставить сообщение достаточно зайти на главную страницу.
